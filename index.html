Toolbar
  • Search – find features
  • Filter – narrow results
  • Edit attributes (table fields)
  • Edit geometry (reshape polygons)
  • Table – view/edit inspection records
  • Layers – toggle parent/related layers
  • Info – open this help
Editing tips
  • Select a site polygon, then use Edit Geometry to drag/add/remove vertices.
  • Open the Site Inspection table to edit inspection fields; save to commit.
  • Relationships are preserved: polygons = parent layer; inspections = related table.
Changes update the live layer/table—double-check before saving.
